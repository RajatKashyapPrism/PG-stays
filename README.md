# PG Stays

Landing page for Premium PG Living — built with [Astro](https://astro.build) and the Prism Design System (`@prism-design-global/components`).

## Stack

- **Astro** — static site framework
- **React** — for interactive library components (via `@astrojs/react`)
- **@prism-design-global/components** — UI component library

## Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Install dependencies                        |
| `npm run dev`     | Start dev server at `localhost:4321`        |
| `npm run build`   | Build production site to `./dist/`          |
| `npm run preview` | Preview production build locally            |
